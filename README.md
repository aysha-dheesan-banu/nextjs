# NextFlow Premium Next.js Template

A premium, high-performance SaaS template built with **Next.js 14**, **App Router**, and **Vanilla CSS** with a focus on glassmorphism aesthetics.

## Features
- ✨ **Stunning Landing Page**: Modern hero section with smooth animations.
- 🔐 **Auth Ready**: Beautifully designed Sign Up and Login pages.
- 📊 **Premium Dashboard**: Data-rich interface with glassmorphism cards and layout.
- 🎨 **Design System**: Harmonious color palette using HSL and CSS variables.
- 📱 **Responsive**: Built to look great on all devices.

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Open the Application**:
   Navigate to [http://localhost:3000](http://localhost:3000)

## Project Structure
- `app/`: Contains the main application routes and layouts.
- `app/globals.css`: The core design system.
- `app/login/`: Login page.
- `app/signup/`: Sign up page.
- `app/dashboard/`: Protected-style dashboard interface.

## Tech Stack
- Next.js 14
- React 18
- Vanilla CSS (Glassmorphism)
- Framer Motion (Optional, configured in package.json)
- Lucide React (Optional, configured in package.json)
